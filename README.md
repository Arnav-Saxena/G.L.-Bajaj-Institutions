#  G.L. Bajaj Institutions — 3D Campus Map  

An interactive 3D map of **G.L. Bajaj Institute**, built completely by hand — optimized for **mobile, tablet, and PC**.  
Every single element, from **balconies to buildings**, was manually modeled and coded.  

 **Live Demo** → [glbajajinstitutions.vercel.app](https://glbajajinstitutions.vercel.app/)  

---

##  Features  
- Fully navigable **3D environment** of the campus  
- **Orbit camera** to freely explore the scene  
- **First-person view** to walk inside the campus  
- **Optimized performance** across devices (PC, tablet, mobile)  
- Completely **handmade** — no auto-generation, no prebuilt assets  

---

## Controls  

### Movement  
- **W / ↑** → Move Forward  
- **A / ←** → Move Left  
- **S / ↓** → Move Backward  
- **D / →** → Move Right  
- **Shift** → Sprint  
- **Space** → Jump
-  **Alt** -> We are not sure what this button does, try it out and let us know.

### Camera  
- **O** → Orbit Camera Mode  
- **P** → First Person Mode  
- **Click** → Lock Cursor (FPS Mode)  

---

##  Tech Stack  
- **Blender** → 3D modeling (campus, details, textures)  
- **Three.js** → Rendering + interactivity  
- **HTML, CSS** → Interface + layout  
- **Vercel** → Deployment
